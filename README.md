# toy-robot
Toy Robot Code Challenge
* Written in Python-3
## Usage
* Test
    `python3 -m unittest tests/test_Robot.py`

* For running at console:

    `python3 run.py` 

* For running with a txt file 

    `./run.py < commands.txt` 

or 

    `chmod a+x run.py` (Assumes \usr\bin\python3 exists)

    ` run.py < commands.txt`
